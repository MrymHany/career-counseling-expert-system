# AI Career Counseling Expert System 🎓🤖

An intelligent decision-support tool designed to provide personalized career recommendations based on 2025–2026 labor market trends.

## ✨ Key Features
- **Trend Integration:** Uses data from the **U.S. Bureau of Labor Statistics (2024-2034)** and **World Economic Forum 2025** reports.
- **Fuzzy Matching:** Implements string matching to align user interests/skills with high-growth job roles.
- **User-Friendly GUI:** Built with Python's Tkinter for an accessible and responsive interface.

## 🛠️ Tech Stack
- **Language:** Python
- **GUI Framework:** Tkinter
- **Logic:** Rule-based expert system with weighted scoring and fuzzy string matching (`difflib`).

## 🚀 Usage
Select your interests, skills, and academic background in the UI. The system will calculate a "Match Score" for various modern career paths and provide a detailed market outlook.
